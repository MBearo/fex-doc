## 倒计时
倒计时组件有三个变量：
| 变量名 | 作用 | 示例 |
| :--- | :--- | :--- |
| 结束时间 | 指定一个时间，该时间减去现在的时间即为计数的时间段 | 2020-10-30 00:00:00
| 倒计时文案 | 倒计时过程中显示在时间前面的文案 | 距离结束还有
| 结束文案 | 倒计时结束之后显示的文案 | 已结束

按照上述表格中的示例，假如您看这篇文章的时间为2020-10-29 00:00:00，则显示：
```
距离结束还有23时59分59秒
```
每秒更新一次，倒计时结束之后，显示：  
```
已结束
```
